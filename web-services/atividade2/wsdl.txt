WSDL significa Web Services Description Language. É o formato padrão para descrever um serviço da web. WSDL foi desenvolvido em conjunto pela Microsoft e IBM.
WSDL é um protocolo baseado em XML para troca de informação em ambientes distribuídos e descentralizados. As definições descrevem como acessar um web service e quais operações vai performar. WSDL é uma linguagem que descreve interfaces de serviçoes baseados em XML, além de ser a linguagem que a UDDI(Universal Description, Discovery, and Integration) usa.
WSDL é frequentemente usado em combinação com SOAP e XML Schema para fornecer serviços da Web pela Internet. Um programa cliente conectado a um serviço da Web pode ler o WSDL para determinar quais funções estão disponíveis no servidor. Quaisquer tipos de dados especiais usados ​​são incorporados no arquivo WSDL na forma de Esquema XML. O cliente pode então usar o SOAP para realmente chamar uma das funções listadas no WSDL.
